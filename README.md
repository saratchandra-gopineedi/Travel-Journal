Travel Journal

React-based travel journal application that presents travel experiences in a clean, card-based interface. This project focuses on practicing core React concepts such as component composition, JSX, props, and modern frontend tooling using Vite.

Live Demo: https://pathnotes.netlify.app/

---

## Features

- Displays travel entries with images, locations, dates, and descriptions
- Component-based architecture using React
- Clean and responsive layout
- Fast development and optimized builds using Vite
- ESLint configured for consistent code quality

---

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
